# 关于

## 这个网站是做什么的？

众所周知，学好数理化，天下都不怕。数理化在手，天下我全有。但是数理化并不是几门很简单的学科，我（们）建立这个网站的目的就是为了让大家能更加轻松地学会、学好数理化，同时也会提供许多教材上没有的拓展知识与联系。

## 我能用这个网站做什么？

您可以查阅并学习相关知识，或者提出意见和建议帮助我们改进。

## 这个网站会更新什么内容

由于这个网站是有一个目前6年级的小学生更新的，所以目前应该是小学奥数以及部分初中内容。如果您有更高的学历，欢迎合作或者贡献。
